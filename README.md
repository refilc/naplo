## Forráskód futtatása

### Projekt klónozása

```sh
git clone --recursive https://github.com/refilc/naplo
cd naplo
```

### Csomagok telepítése

Futtasd a `fix-pub.sh` nevű fájlt.

### Alkalmazás futtatása

```sh
cd filcnaplo
flutter run
```
